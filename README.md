Ecommerce Sales Performance Dashboard – Power BI

## Overview
Developed an interactive Power BI dashboard to monitor and analyze 
ecommerce sales performance across regions, product categories, 
and time periods to support data-driven business decisions.

## File
- NEHA ECOMMERCE SALES DASHBOARD.pbix — Power BI dashboard file

## Tools Used
Power BI — DAX, Power Query, Star Schema, Slicers, 
Line Charts, Bar Graphs, KPI Cards

## Dashboard Features
- Monthly sales trend analysis
- Region-wise and category-wise performance breakdown  
- Top-performing products identification
- Star Schema data model for optimized reporting
- Dynamic slicers for interactive filtering

## Data Preparation
- End-to-end data cleaning using Power Query
- Data transformation and type formatting
- Fact and Dimension table relationships
